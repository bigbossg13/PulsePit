# PulsePit

FRC & FTC Resource Hub — development in progress.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the hub.

## Adding Content

Create a `.mdx` file in the appropriate `content/` subdirectory with the required frontmatter. See `/about` in the running app for the full schema.
